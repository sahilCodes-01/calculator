# calculator

Features

Performs basic arithmetic operations: Addition, Subtraction, Multiplication, and Division

Handles continuous calculations without needing to press equals each time

Displays results instantly using dynamic DOM updates

Automatically starts a new calculation after result display

Prevents invalid operations like division by zero with error handling

Includes a Clear button to reset calculator state and display

Maintains calculation flow using internal state management

Supports operator switching before entering the second number

Clean event-listener based input handling for digits, operators, equals, and clear actions

Resets calculator automatically when an error occurs
